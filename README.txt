An implementation of Minecraft 1.5.2 NBT decoder built from scratch to counter
memory exhaustion exploit achievable by specially crafted payloads.

SafeBT is fully compliant with Minecraft NBT decoder implementation.
Any NBT payloads generated by default implementation of NBT are decoded properly.

Exploit and explanation - https://archive.ph/ipNCH
Malicious payloads for testing - see src/test/resources/nbt
